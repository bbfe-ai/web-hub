# 项目路线图 (Roadmap)

本文档列出 WebHub 的开发计划和未来方向。

## 📅 2026 年计划

### Q2 (4-6月) - 基础完善

#### ✅ 已完成
- [x] 核心项目管理功能
- [x] 多截图支持（最多4张）
- [x] 三种截图方式（自动/手动/粘贴）
- [x] 双视图切换（网格/列表）
- [x] 分类管理和搜索
- [x] iframe 内嵌浏览
- [x] 拖拽导入 URL
- [x] 富文本描述编辑器
- [x] Docker 容器化部署
- [x] 完整的 README 文档
- [x] GitHub 开源配置

#### 🔄 进行中
- [ ] 用户认证系统（基础 HTTP Auth）
- [ ] 深色主题支持
- [ ] 移动端响应式适配

#### 📋 待开始
- [ ] 项目访问统计
- [ ] 截图自动更新定时任务
- [ ] 数据导出/导入功能（JSON/CSV）

### Q3 (7-9月) - 功能增强

- [ ] **用户系统**
  - 多用户支持
  - 角色权限管理（管理员/普通用户）
  - 用户个人项目空间
  
- [ ] **项目组织**
  - 项目分组/文件夹功能
  - 项目标签系统（多标签）
  - 项目收藏和置顶
  
- [ ] **截图增强**
  - 截图对比功能
  - 截图标注工具
  - 截图历史记录
  
- [ ] **浏览器集成**
  - Chrome 扩展程序
  - Firefox 扩展程序
  - 浏览器书签同步

### Q4 (10-12月) - 高级功能

- [ ] **API 增强**
  - RESTful API 完整覆盖
  - API 密钥认证
  - Webhook 支持
  
- [ ] **自动化**
  - 定时截图任务
  - 项目健康检查
  - 自动备份机制
  
- [ ] **监控告警**
  - 项目可用性监控
  - 异常通知（邮件/钉钉/企业微信）
  - 性能指标收集
  
- [ ] **插件系统**
  - 插件架构设计
  - 常用插件开发
  - 插件市场

---

## 🎯 长期愿景

### 2027 年目标

1. **企业级特性**
   - SSO 单点登录集成（OAuth2/SAML）
   - LDAP/AD 目录集成
   - 审计日志和操作记录
   - 数据加密存储

2. **云原生支持**
   - Kubernetes Helm Chart
   - 多云部署支持
   - 自动扩缩容
   - 服务网格集成

3. **AI 增强**
   - 智能项目分类
   - 截图内容识别
   - 异常检测
   - 自然语言搜索

4. **生态系统**
   - 官方插件市场
   - 第三方集成（Jira/GitLab 等）
   - API SDK（Python/Go/Java）
   - 开发者社区建设

## 💬 参与规划

我们欢迎社区参与路线图的制定！如果你有以下想法：

- ✨ 新功能建议
- 🔧 现有功能改进
- 🐛 优先级调整
- 📊 新的使用场景

请通过以下方式参与：

1. **创建 Issue**：使用 [Feature Request](https://github.com/your-username/web-hub/issues/new?template=feature_request.md) 模板
2. **参与讨论**：在 [GitHub Discussions](https://github.com/your-username/web-hub/discussions) 中发表观点
3. **投票**：为重要的功能点赞 👍
4. **贡献代码**：实现你期待的功能
